# Installation

## Requirements

- Python 3+
- Pipenv
- Django 1.11+

## Steps

```
apt-get install python3 python3-pip
pip install pipenv
git clone https://github.com/cdubz/first-django-app.git
cd first-django-app/
pipenv install --tree
pipenv shell
cd mysite
python manage.py runserver
```
